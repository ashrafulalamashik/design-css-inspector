# Design & CSS Inspector Chrome Extension

A powerful Manifest V3 Chrome Extension that allows developers and designers to instantly inspect CSS properties, extract typography, find semantic color palettes, and check WCAG contrast ratios with a simple click.

## Features
* **Point-and-Click Inspector:** Instantly view CSS properties of any element.
* **Typography & Color Extraction:** Automatically pull fonts and color palettes from the active tab.
* **Asset Explorer:** Extract all images and SVGs.
* **Contrast Scanner:** Check text accessibility against backgrounds.

## Installation Guide (For Users)
Since this extension is not yet published on the Chrome Web Store, you can easily install it locally using Developer Mode.

1. **Download the Code:** Click the green **Code** button and select **Download ZIP**, then extract the ZIP file on your computer.
2. **Open Extensions Page:** Open Google Chrome and navigate to `chrome://extensions/` in the URL bar.
3. **Enable Developer Mode:** Turn on the **Developer mode** toggle switch in the top right corner.
4. **Load the Extension:** Click the **Load unpacked** button in the top left.
5. **Select Folder:** Select the extracted folder that contains the `manifest.json` file.
6. **Done!** Pin the extension to your toolbar and start using it.
